# Rock-Paper-Scissors-Game
A custom, responsive Rock, Paper, Scissors game built to challenge friends, track scores in real-time, and settle friendly rivalries. Features automated robot matchmaking, state management, and an adaptive layout for desktop and mobile play
## 🛠️ Features
* **Smart Device Detection:** Play using keyboard shortcuts (`S` to start, `R` to restart) on desktop, or intuitive tap controls on mobile screens.
* **Persistent Score Tracking:** Keeps exact counts of human vs. robot victories across matches.
* **State & Timing Control:** Uses asynchronous JavaScript timers to manage round transitions smoothly.

## 💻 Tech Stack
* **HTML5:** Semantic architecture with inputs utilized as real-time UI displays.
* **CSS3:** Responsive layout styling for centering game boards across devices.
* **JavaScript (ES6):** Custom game loops, event listener management, and DOM manipulation.
